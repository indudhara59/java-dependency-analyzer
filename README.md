# Java Dependency Analyzer

A Java-based tool for analyzing class-level and method-level dependencies in Java projects.

## Project Status

Work in progress.

## Goal

The goal of this project is to identify dependencies between classes and methods in Java code and represent them as dependency graphs.

## Technologies

- Java
- Maven
- Graph Analysis

## Planned Features

- Analyze class-level dependencies
- Analyze method-level dependencies
- Generate dependency graphs
- Identify coupling between classes
- Help understand software architecture patterns

## Current Focus

Initial project setup and dependency extraction design.
